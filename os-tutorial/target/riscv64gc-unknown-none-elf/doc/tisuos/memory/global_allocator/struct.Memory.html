Memory

内存管理中最重要的结构 将申请的页表分割成 2 的幂次大小，记录数量和起始地址 每个结构体以链表的形式连接，根据分割粒度的大小排序方便查找 每个颗粒的分配情况通过位图记录，即一位一个颗粒，已用的标记为 1 search idx 是辅助搜索的位点，具体可以查看 alloc 和 free 的逻辑 Page 记录当前管理内存的属性
